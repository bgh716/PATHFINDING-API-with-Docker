  • MAZE GUIDE
  • 1. Press start button to set start position and goal position(random).
  • 2. Build a maze as clicking the board. Black color means that the area is the wall now(initial walls are built at the first time. You can modify it).
  • (if there is no path between start position and goal position, "no path" alert will pop up)
  • 3. Press find button to find the shortest paths.
  • 5. Press F5 to reset.
